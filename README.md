# Luna16Project

This repository contains the project for the BME-515 class.

## Files

- `dataset/`: Directory containing the dataset.
- `Luna16Project/`: Directory containing the project source code.
- `BME-515 FInal Report.pdf`: The final report for the project.
- `Copyright.txt`: Copyright information.
- `How To Run.docx`: Instructions on how to run the project.
- `Inferences.html`: HTML output of inferences.
- `Inferences.pdf`: PDF output of inferences.
- `Luna_Inference.pdf`: PDF of Luna inference.
